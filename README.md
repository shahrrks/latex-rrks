# Shah Rrks' LaTeX Snippets

This extension provides a collection of LaTeX snippets for a more efficient and streamlined writing experience. These snippets are a compilation of general-purpose templates and shortcuts used by Shah Rrks in their daily LaTeX writing tasks.

## Features

- A set of handy LaTeX snippets for common structures and commands.
- Streamlined insertion of frequently used templates with just a few keystrokes.

## Snippets Included

This is the first version of Shah Rrks' LaTeX Snippets extension, and it includes a variety of general snippets for writing LaTeX text. 

## Usage

1. Install the extension.
2. Open a LaTeX file.
3. Type the prefix of the desired snippet, and select the snippet from the IntelliSense suggestions.
4. The snippet will auto-expand, and you can fill in any placeholders.
